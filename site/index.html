02 — THE COMMAND BAR

Build-a-bar

Pick scripts → get a dockable AE panel bundle
Check the tools you want — get a ready-to-dock bundle. The zip contains the dockable IVGD Command Bar panel plus your scripts in an ivg-scripts subfolder. Drop both into ScriptUI Panels: only the command bar appears in AE's Window menu, and its buttons reflow when docked vertical, horizontal, or as a grid.
0 of 32 selected Prefer the menus? Any .jsx from the bundle also works dropped straight into the Scripts folder.
03 — GET STARTED

Install in three steps

01 Download Grab the .jsx for the tool you need — or the full .zip of all 32.
02 Drop it in Move it to your After Effects ▸ Scripts folder, or use File ▸ Scripts ▸ Run Script File.
03 Run it Launch from the Scripts menu (or ScriptUI Panels for dockable tools). Done.
