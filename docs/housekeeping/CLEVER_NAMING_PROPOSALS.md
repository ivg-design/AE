# Script Naming Convention & Proposals

## Naming Convention

This repository uses creative suffixes inspired by existing names like "Rectangulator" and "Guiderator":

- **-ator**: Generators and automation tools
- **-izer**: Transform/modify tools
- **-tron**: Precise control tools
- **-matic**: Automatic processing
- **-master**: Comprehensive control suites
- **-mate**: Helper/companion tools
- **-forge**: Build/construct tools
- **-smith**: Precision crafting
- **-pilot**: Navigation tools
- **-sync**: Synchronization tools

## Naming Highlights

### Best Original Names (Keep As-Is)

1. **Rectangulator** - Iconic, creative, memorable
2. **Guiderator** - Perfect -ator usage
3. **celMate** - Brilliant wordplay (cel animation + mate)
4. **keyMate** - Consistent with celMate
5. **sfxMaster** - Clear domain mastery

### Most Creative New Names

1. **ChromaBlender** - Evocative color tool name
2. **PathPilot** - Navigation metaphor
3. **LimbForge** - Strong creation imagery
4. **TimeWarpTron** - Sci-fi time control
5. **VertexMaster** - Technical mastery
6. **NullMatic** - Automatic null creation
7. **OriginSmith** - Craftsmanship metaphor
8. **PathStalker** - Playful following metaphor
9. **RandoMatic** - Fun randomization name
10. **DualParentron** - Electronic dual control

### Naming Patterns Used

- **-Tron**: For precise, electronic control (BurstTron, TimeWarpTron, TrimPathTron)
- **-Matic**: For automatic operations (NullMatic, RandoMatic, KeyCloneMatic, FrameMatic)
- **-Forge**: For creation tools (LimbForge, MultiParentForge, SlideForge, SubtitleForge)
- **-Master**: For comprehensive control (VertexMaster, sfxMaster)
- **-Pilot**: For navigation tools (PathPilot, FramePilot)
- **-ator**: For generators (Guiderator, Rectangulator, PathCentrator)
- **-Smith**: For precision crafting (OriginSmith)
